# Health Care App

## 概要

本アプリは、日々の健康状態（血圧・脈拍・体重）を記録し、BMIの自動計算や過去データの可視化を行うAndroidアプリです。
ユーザーが自身の健康状態を継続的に管理しやすいよう、シンプルかつ直感的なUIを意識して設計しました。

---

## 開発背景

近年、健康管理の重要性が高まる中で、日々の体調を手軽に記録・振り返ることができるアプリの必要性を感じ、本アプリを開発しました。
特に、数値を記録するだけでなく「変化を視覚的に理解できる」ことを重視し、グラフ機能の実装を行いました。

---

## 主な機能

* 健康データの記録（最高血圧・最低血圧・脈拍・体重・身長）
* BMIの自動計算
* 履歴一覧表示（RecyclerView）
* グラフによるデータの可視化
* 日付ごとのデータ管理
* データの永続化（ローカル保存）

---

## 使用技術

* 言語：Kotlin
* 開発環境：Android Studio
* データベース：Realm
* アーキテクチャ：MVVM
* UI：Jetpack（ViewModel / LiveData など）

---

## 工夫した点

* **MVVMアーキテクチャの採用**
  UIとロジックを分離することで、保守性・拡張性の高い設計を実現しました。

* **データのリアルタイム更新**
  RealmとViewModelを組み合わせることで、データ変更時にUIへ即時反映される仕組みを構築しました。

* **BMIの自動算出機能**
  入力された身長・体重から自動でBMIを計算し、ユーザーの負担を軽減しました。

* **視覚的に分かりやすいグラフ表示**
  数値の変化を直感的に把握できるよう、折れ線グラフでデータを表示しました。
